【手牌 / hand】
【規則說明 / Rule】
如何獲勝 / How to Win?
打完所有手牌你就獲勝了！

打牌規則 / Rule to Play Card
‧每堆牌堆只要達7種顏色就會消失，消失後會從牌組中補一張牌到該牌堆
‧如果打出的牌有牌堆中顏色，抽取「重複顏色種類數」一半的牌，無條件捨去(打出奇數重複受到懲罰較小)
‧無論抽牌或打牌都會增加行動數，用越低的行動數打完手牌即表示打得越好

操作 / Manipulation
點擊場面牌(出現紅框表示選取) -> 點擊手牌 -> 手牌移至該牌堆
