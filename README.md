**規則**
![Alt text](/Source/play1.png)
- 上圖為初始版面
- 獲勝方式為打完所有手牌
- 操作 : 點擊場面牌(出現紅眶表示選取) -> 點擊手牌 -> 手牌移至該牌堆
- 每堆牌堆只要達7種顏色就會消失，消失後會從牌組中補一張牌到該牌堆
- 懲罰 : 如果打出的牌有牌堆中顏色，抽取「重複顏色種類數」一半的牌，無條件捨去(打出奇數重複受到懲罰較小)
- 無論抽牌或打牌都會增加行動數，用越低的行動數打完手牌即表示打得越好
- 獲勝!!
